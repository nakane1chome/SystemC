Detailed Description

A sc_uint is an unsigned integer whose length is less than.

the machine's native integer length. We provide two implementations (i) sc_uint with length between 1 - 64, and (ii) sc_uint with length between 1 - 32. Implementation (i) is the default implementation, while implementation (ii) can be used only if compiled with -D_32BIT_. Unlike arbitrary precision, arithmetic and bitwise operations are performed using the native types (hence capped at 32/64 bits). The sc_uint integer is useful when the user does not need arbitrary precision and the performance is superior to sc_bigint/sc_biguint.

Original Author: Amit Rao, Synopsys, Inc.
