160 #ifdef SC_MAX_NBITS
161 extern const int MAX_NDIGITS;
162 // Consider a number with x bits another with y bits. The maximum
163 // number of bits happens when we multiply them. The result will have
164 // (x + y) bits. Assume that x + y <= SC_MAX_NBITS. Then, DIV_CEIL(x) +
165 // DIV_CEIL(y) <= DIV_CEIL(SC_MAX_NBITS) + 2. This is the reason for +2
166 // above. With this change, MAX_NDIGITS must be enough to hold the
167 // result of any operation.
